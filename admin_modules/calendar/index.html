About this Module

A team-wide event planner for scheduling company releases, project milestones, deadlines, and team gatherings. Includes a monthly calendar view.

Key Actions

  • Add events with date, location/virtual link, and organizer.
  • Open the monthly calendar view to browse events by day.
  • Search events by title, location, or date.
  • Edit or delete any scheduled event entry.
