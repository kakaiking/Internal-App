Google OAuth Setup

Enter your Google Client ID to enable real authentication.

Setup Instructions:
  1. Go to the Google Cloud Console.
  2. Create or select a project.
  3. Go to APIs & Services > OAuth consent screen and configure it.
  4. Go to Credentials > Create Credentials > OAuth client ID.
  5. Select Web application as the application type.
  6. Add this URI under Authorized JavaScript origins:
    http://localhost:3000
  7. Copy your Client ID and paste it below.
