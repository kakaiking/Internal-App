About this Module

An encrypted inbox for secure team announcements, confidential notifications, and credential transmissions. All messages are encrypted with a pre-shared key.

Key Actions

  • Compose and transmit encrypted messages to the team.
  • Enter a shared decryption key to unlock and read the real-time feed.
  • View Top Senders leaderboard and search the ciphertext stream.
  • Clear the entire message stream when no longer needed.
