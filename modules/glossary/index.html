About this Module

An internal dictionary defining business acronyms, technical jargon, team codenames, and terminology to ensure everyone speaks the same language.

Key Actions

  • Add new terms with definitions and contributor attribution.
  • Browse alphabetically using the A–Z filter bar.
  • Search terms by keyword across definitions and titles.
  • View the Top Contributors leaderboard; edit or delete entries.
